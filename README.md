# node-demo
develop webservice on node server wiht a most simple mode
### Project setup
```
npm install
```
### add env file for env-var
```
touch .env
```
### add the env vars
```
HTTP_PORT = "xxx"
```
### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

###  Lints and fixes files
```
npm run lint
```
